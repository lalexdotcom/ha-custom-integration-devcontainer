# HA Custom Integration

[!WARNING]
All files outside of the .devcontainer could be overwritten by the template (including this README)

## prerequisites
This template is a devcontainer for VS Code. For more information, see the [VSCode docs](https://code.visualstudio.com/docs/devcontainers/containers)

## Install
Fork this repository, then use it as a template for your new awesome project. (see [Github doc](https://docs.github.com/fr/repositories/creating-and-managing-repositories/creating-a-repository-from-a-template))
Then clone tour repository with VS Code, reopen it in container, answer to prompts and let the magic happen! :sparkles: :sparkles: :sparkles:
